# 🍳 Recipe Finder App

A powerful, user-friendly recipe search web application that lets you discover dishes from around the world. Just type any food name and get instant recipes with ingredients, step-by-step instructions, and YouTube tutorials!

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔍 **Smart Search** | Search any dish from any cuisine worldwide |
| 🌍 **Global Coverage** | Pakistani, Indian, Italian, Chinese, Mexican & more |
| 🎲 **Random Backup** | If no match found, get a random recipe suggestion |
| 📱 **Responsive Design** | Works perfectly on mobile, tablet & desktop |
| 🖼️ **Recipe Cards** | Beautiful grid view with dish images |
| 📖 **Detailed Modal** | Full ingredients + step-by-step instructions |
| ▶️ **YouTube Integration** | Watch cooking tutorials directly |
| 🎯 **Quick Suggestions** | Click popular dishes to search instantly |
| ⚡ **Fast & Lightweight** | No frameworks, pure vanilla JS |

## 🚀 How to Use

1. Type any dish name in the search bar
   - Examples: `Biryani`, `Pizza`, `Pasta`, `Karahi`, `Sushi`
2. Press **Enter** or click the **Search** button
3. Browse recipe cards with dish images
4. Click any card to view full recipe details
5. View ingredients list & step-by-step instructions
6. Click YouTube link to watch cooking tutorial
